## Run Locally

Clone the project

```bash
  git clone https://github.com/thaoanhhaa1/ReactNative_Youtube.git
```

Go to the project directory

```bash
  cd ReactNative_Youtube
```

Install dependencies

```bash
  npm install
```

Start project

```bash
  npm start
```
